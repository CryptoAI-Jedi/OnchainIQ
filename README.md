# OnchainIQ

OnchainIQ is a natural-language onchain intelligence agent: ask questions about
public blockchain and Bitcoin-mining activity in plain English, and an AI agent
retrieves the right pre-built query, runs it against public data (Dune Analytics),
and returns a grounded, sourced answer with a chart. Built in the open.
Public blockchain data only. Informational and analytical tool, not financial advice.
